AFK API

AFKを設定, 取得します。
認証： ユーザーID APIキー（sbuauthと呼びます）
スコープ： afk

AFKを取得 - GET /afk

パラメータ

名前 タイプ 説明 省略 デフォルト 例
userid 数値 取得したいユーザーID。 不可 - 718760319207473152

戻り値

名前 タイプ 説明 例
message 真偽型 指定したユーザーがAFKかどうか False
reason? 文字列 指定したユーザーのAFKの理由 "眠い"

AFKを設定 - POST/PUT /afk

パラメータ

名前 タイプ 説明 省略 デフォルト 例
reason 文字列 AFKの理由を指定します。 可能 理由無し "眠い"

戻り値

名前 タイプ 説明 例
tweet null|false|文字列(URL) ツイートのURL。失敗時はfalse, 未登録時はnull "https://twitter.com/..."

AFKを解除 - DELETE /afk

パラメータ

(無し)

戻り値

名前 タイプ 説明 例
urls 配列[文字列(URL)] メンションされたメッセージのURL。 ["https://discord.com/channels/...."]
tweet null|false|文字列(URL) ツイートのURL。失敗時はfalse, 未登録時はnull "https://twitter.com/..."
